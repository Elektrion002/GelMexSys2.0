1. Módulo de Ventas y Preventa

¿QUÉ ES?

Es la **Consola de Control Comercial** donde se capturan los pedidos de preventa. Es el primer punto de contacto digital entre el cliente y la operación de GelMex.

¿POR QUÉ EXISTE?

Para eliminar el error humano en la toma de pedidos, asegurar el cobro correcto según el perfil del cliente y proteger las finanzas mediante validaciones de crédito.

¿PARA QUÉ SIRVE?
  • Validación de Crédito: Bloquea ventas si el cliente excede su límite (Deuda + Pedidos vivos).
  • Precios VIP: Aplica automáticamente los descuentos pactados.
  • Trazabilidad: Genera folios únicos que activan producción.
¿CÓMO SE USA?
  1. Accede a Ventas -> Preventa.
  2. Selecciona al Cliente (el sistema cargará su estado financiero).
  3. Agrega los productos y cantidades al carrito.
  4. Define la Fecha de Entrega y el método de pago.
  5. Escribe una nota detallada (mín. 15 caracteres) y guarda.

2. Catálogo Maestro de Productos

La "Ingeniería de Producto" es el cerebro del sistema. Aquí se definen los estándares que rigen toda la operación.

Identidad Única (SKU)

Cada producto debe tener un **SKU único**. El sistema lo convierte automáticamente a mayúsculas para mantener el orden. Ejemplo: `PAL-FRE-01`.

Inteligencia de Precios

Al ingresar el **Precio de Costo** y el **Precio de Venta**, el sistema calcula instantáneamente el **Margen de Utilidad (%)** para toma de decisiones rápidas.

Estándar de Imagen

Para una experiencia premium en el portal de clientes, las fotos deben ser de alta calidad y fondo neutro. El sistema las optimiza y renombra automáticamente usando el SKU.

3. Infraestructura y Catálogos

GelMexSys utiliza un sistema de **Catálogos Dinámicos** para gestionar la base del negocio.

El Explorador de Infraestructura

Usa esta herramienta visual para navegar en cascada:

  • Plantas: Unidades de negocio principales.
  • Áreas: Departamentos dentro de la planta (Producción, Ventas, etc).
  • Almacenes: Puntos físicos de resguardo.
  • Ubicaciones: El estante o rack exacto donde vive el producto.
Nota de Seguridad

La edición de catálogos requiere **Nivel 4 (Gerencia)**. Cambios en unidades de medida o SKUs pueden afectar registros históricos de ventas.
