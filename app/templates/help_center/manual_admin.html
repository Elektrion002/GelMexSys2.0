2. Gestión de Productos

El núcleo del sistema. Aquí registras la cara de GelMex al cliente.

  • SKU Único: Cada producto debe tener un código irrepetible.
  • Precios: El Precio Venta General es la base, pero el sistema prioriza los Precios Especiales por cliente en el portal.
  • Imagen: Sube fotos nítidas. Si falta una, el sistema asignará el logo de GelMex automáticamente.
